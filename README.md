# Trabajo Práctico Nº3 Sistemas Operativos

### Instructions to compile and execute
To compile program

```bash 
    make clean
    make all
```

To execute program

To run the server:
```sh
./server
```

To run the client:
```sh
./client
```

To clean everything
```bash 
    make clean
```
